Vilcom Ops Network An interactive 3D map for visualizing network coverage and infrastructure.

Tech Stack: React, Deck.gl (WebGL), Turf.js, Lucide React.

Key Features:

Global to Local Zoom: Seamless transition from world view to county-level granularity.

Data Visualization: Renders 3D extruded terrains, curved network links (ArcLayer), and site clusters (ScatterplotLayer).

UX/UI: Custom trailing cursor, auto-rotation idle states, and floating camera controls.

Geospatial Logic: Real-time distance calculation and centroid targeting for Kenyan counties.
